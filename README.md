# RpgNaming

RPGに出てきそうな名前をランダムで返してくれるツールです。

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rpg_naming'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rpg_naming

## Usage

```
RpgNaming.monster
#=> "ワーウルフ"
```
